Tablas

Con los TAG`s TABLE, TR, TH y TD se crean tablas que organizan informacion para resumirla.

  • TR: permite declarar renglones
  • TH: permite declarar cabeceras de columna
  • TD: para declarar los datos de las celdas

Los artributos: border, caption,colspan y rowspan

  • BORDER: Establece el ancho de la tabla
  • CAPTION: Establece un titulo arriba de la cabecera
  • COLSPAN: Permite que una celda ocupe varias columnas
  • ROWSPAN: Permite que una celda ocupe varios renglones
