Links e imagenes

img

<A> Es un tag para inscrustar imagenes, sus atributos principales son:

  • title
  • href
  • target

Opciones de target:
  • SELF: Abre el link en la misma pagina
  • BLANK: Abre el link en una nueva pagina del navegador
  • PARENT_ Abre en ña pagina contene
ejemplo: Sitio a github

Imagenes

El tag <IMG> Sirve para incrustar imagenes en la pagina.

Sus atributos importantes son:

  • Title: Es el tooltip
  • src: Es la URL a la imagen, ya sea relativa o absoluta
  • width: Ancho en pixeles
  • hight: Alto en pixeles
  • loading: forma de carga: Lazy o eager.
